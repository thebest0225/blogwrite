쿠션글 생성

1) 유입시킬 목적지글 (발행된 글)
2) 쿠션으로 쓸 계정 (체크한 수만큼 각기 다른 글 생성)

※ 각 쿠션글은 목적지글로 '자세히 보기' 유입 링크가 자동으로 들어갑니다. 생성 후 작업보드에서 검토·발행하세요.
