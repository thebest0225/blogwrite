# 구글 시트 연동 설정 (작업 기록 · 내 글 보관함 클라우드)

확장이 **발행/생성할 때마다 구글 시트에 자동 기록**하고, **링크 걸 때 그 시트에서 내 글을 먼저 찾아** 연결하도록 하는 설정입니다. 서버 없이 무료로 됩니다.

## 1. 구글 시트 만들기
1. https://sheets.google.com 에서 **새 시트** 생성 (이름 아무거나, 예: "블로그기록")

## 2. Apps Script 붙여넣기
1. 그 시트에서 상단 메뉴 **확장 프로그램 → Apps Script**
2. 기본 코드 다 지우고 아래 코드를 붙여넣기:

```javascript
// 필요하면 토큰을 정하세요(확장 설정의 '비밀 토큰'과 동일하게). 안 쓰면 "" 로.
const TOKEN = "";

function sheet_() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  let sh = ss.getSheetByName("records");
  if (!sh) {
    sh = ss.insertSheet("records");
    sh.appendRow(["date", "type", "title", "url", "keyword", "summary", "platform", "images", "body"]);
  }
  return sh;
}

function json_(obj) {
  return ContentService.createTextOutput(JSON.stringify(obj)).setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  try {
    const d = JSON.parse(e.postData.contents || "{}");
    if (TOKEN && d.token !== TOKEN) return json_({ ok: false, error: "bad token" });
    sheet_().appendRow([
      new Date(), d.type || "", d.title || "", d.url || "", d.keyword || "",
      d.summary || "", d.platform || "", (d.images || []).join(" "), d.body || ""
    ]);
    return json_({ ok: true });
  } catch (err) {
    return json_({ ok: false, error: String(err) });
  }
}

function doGet(e) {
  if (TOKEN && (e.parameter.token || "") !== TOKEN) return json_({ records: [] });
  const sh = sheet_();
  const values = sh.getDataRange().getValues();
  const rows = values.slice(1).map(function (r) {
    return { date: r[0], type: r[1], title: r[2], url: r[3], keyword: r[4], summary: r[5], platform: r[6], images: r[7], body: r[8] };
  });
  return json_({ records: rows.reverse() });
}
```

3. (선택) 비밀 토큰을 쓰려면 위 `const TOKEN = "";` 에 아무 문자열을 넣으세요.

## 3. 웹앱으로 배포
1. 우측 상단 **배포 → 새 배포**
2. 유형(톱니바퀴) → **웹 앱** 선택
3. 설정:
   - 실행: **나(본인)**
   - 액세스 권한: **모든 사용자** (또는 "링크가 있는 모든 사용자")
4. **배포** → 권한 승인(본인 구글 계정)
5. 나오는 **웹 앱 URL**(`https://script.google.com/macros/s/..../exec`) **복사**

## 4. 확장에 입력
1. 확장 **설정 → ☁️ 구글 시트 연동**
2. **Apps Script 웹앱 URL** 붙여넣기 (+ 토큰 썼으면 토큰도)
3. 저장

## 이제 이렇게 동작합니다
- 글 **발행/생성**할 때마다 시트 `records` 탭에 한 줄씩 자동 기록(제목·URL·키워드·요약·본문 등)
- **워드프레스 발행**이나 **메인 블로거 URL 입력** 시 그 글이 **내 글 보관함**에 저장(시트에도)
- **링크 걸 때** 이 보관함(시트+로컬)에서 관련 내 글을 **먼저 찾아** 연결 → 여러 글이 서로 링크로 엮임
- 여러 PC/브라우저에서 같은 시트를 봐서 **기록이 안 사라지고 동기화**

## 참고
- 코드를 수정하면 **배포 → 배포 관리 → 편집(연필) → 새 버전**으로 재배포해야 반영됩니다.
- 시트를 직접 열어 기록을 눈으로 확인/편집할 수 있습니다.
- 보관함 삭제(✕)는 로컬 기준입니다(시트 행은 시트에서 직접 지우세요).
