# 블로그 오토라이터 v0.2 — 설치 & 사용법

## 1. 설치
1. `chrome://extensions` → 우측 상단 **개발자 모드 ON**
2. **압축해제된 확장 프로그램을 로드** → `영상오토\extension` 폴더 선택
3. 코드 수정하면 목록에서 **새로고침(↻)**

## 2. 설정 (⚙️)
- **KIE API 키** (필수) — 글·이미지 (kie.ai/api-key)
- **글 모델**: Claude Sonnet 5 / Opus 4.8
- **내 블로그 주소** — 링크 우선 연결 대상
- (선택) **워드프레스 REST** — 직접 발행용
- (선택) **구글 시트 연동** — 작업 기록·보관함 클라우드 (설정_구글시트연동.md)
- (선택) **애드센스 광고 코드**, **저자 소개**, **톤/독자 기본값**

## 3. 사용 흐름 (핵심)
> 정확한 원본은 **클로드(claude.ai, 웹서치)** 로 쓰고, 확장은 **디자인·보강·3종 변형·썸네일·링크**를 담당.

1. **클로드에서 원본 글 작성** (출처·링크 포함, 텍스트 또는 HTML)
2. 확장 사이드패널 **원본 글** 칸에 붙여넣기
3. 상단 탭 선택:
   - **블로거 메인**: 원형 + 보강 + 디자인 (내 메인 글)
   - **네이버 / 워드프레스**: 쿠션 글 — 링크 절반은 메인 블로거로 유입, 클릭 거리 많게
4. (네이버/워드프레스면) **메인 블로거 글 URL** 입력
5. 이미지 옵션: 생성 on/off · 개수(기본 1=썸네일만) · 비율(모드별 자동: 블로거·WP 4:3, 네이버 1:1)
6. **생성** → 미리보기
7. **📋 HTML 복사** → 블로거/네이버/워드프레스 'HTML 편집'에 붙여넣기
   (워드프레스는 **WP 바로발행/초안** 버튼으로 바로도 가능)

## 4. 편의 기능
- **🔥 트렌드**: 구글 급상승 검색어(6시간 자동 갱신) → 클릭하면 주제(키워드)로. 주제 선정용.
- **🔗 내 글 보관함**: 발행/메인URL 입력 시 자동 저장 + 수동 추가. **링크 걸 때 여기서 먼저 찾아** 연결(웹서치 대체). 구글 시트 연동 시 클라우드 동기화.
- **🖌️ 이미지 수정**: 생성된 이미지마다 "다시 생성"(프롬프트 반영) / "부분 수정"(image-to-image).

## 5. 참고
- 링크는 문구 뜻에 맞는 실제 목적지로 감(넷플릭스 억측 X → 네이버 "○○ 다시보기" 등). 내 보관 글·원본 링크가 있으면 그걸 우선.
- 사실 정확도는 클로드 원본이 책임짐(KIE는 재가공만). 애드센스 정책상 낚시성 클릭유도 문구는 넣지 않음.
- 네이버는 편집기 자동입력 미지원 → HTML 복사로 붙여넣기.
